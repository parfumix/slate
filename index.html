    Introduction

    Welcome to the HUB API! You can use our API to access HUB API endpoints, which can get information on various tables in our database.

    We have language bindings in Shell, Ruby, Python, and JavaScript! You can view code examples in the dark area to the right, and you can switch the programming language of the examples with the tabs in the top right.

    Authentication

    To authorize, use this code:

    const http = require('http');

let api = http.authorize('access_token');

    Make sure to replace acess_token with your API key.

    HUB expects for the API key to be included in all API requests to the server in a header that looks like the following:

    Authorization: access_token

    You must replace access_token with your personal API key.

    Errors

    The HUB API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request is invalid.
    401 Unauthorized -- Your API key is wrong.
    403 Forbidden -- The project requested is hidden for administrators only.
    404 Not Found -- The specified project could not be found.
    405 Method Not Allowed -- You tried to access a project with an invalid method.
    406 Not Acceptable -- You requested a format that isn't json.
    410 Gone -- The project requested has been removed from our servers.
    418 I'm a teapot.
    429 Too Many Requests -- You're requesting too many projects! Slow down!
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
